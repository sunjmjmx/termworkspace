=== 场景：文件浏览器 ===
文件：05-file-browser.png（待补充真实截图）

触发方式：应用启动后，左侧自动显示文件树

界面布局：
```
src/
├── main/
│   ├── platform.ts     ← 点击此文件
│   └── index.ts
├── renderer/
│   ├── components/
│   ├── App.tsx
│   └── index.tsx
├── preload/
│   └── index.cjs
└── types/
    └── index.ts
```

交互行为：
- 点击文件：终端输出文件完整路径（如 /Users/user/project/src/main/platform.ts）
- 点击文件夹：展开/折叠
- 文件树左上角有折叠按钮（←），可收起给终端腾空间

代码路径：src/renderer/components/FileTree.tsx

测试命令：启动后点击左侧文件树中的任意文件
测试结果：✓ 文件路径显示在终端标题栏
