=== 场景：主界面全貌（多 Tab + 终端） ===
文件：01-main-interface.png（待补充真实截图）

触发方式：npm start 启动应用后，自动显示主窗口

界面布局：
```
┌──────────────────────────────────────────────┐
│  ~/projects/termworkspace  Tab1 │ Tab2 │ + │🌙│
├──────────┬───────────────────────────────────┤
│           │  $ npm run dev                    │
│  src/     │  > vite v6.0.0 ready in 320ms    │
│  docs/    │                                   │
│  tests/   │                                   │
│  .env     │                                   │
│           │                                   │
└──────────┴───────────────────────────────────┘
```

元素构成：
- 顶部 TabBar：项目路径（〜/projects/termworkspace）+ 两个 Tab 标签 + 新建按钮(+) + 主题切换按钮(🌙)
- 左侧文件树：项目目录结构，可折叠
- 主区域：xterm.js 终端，显示命令行提示符

当前终端已自动 cd 到项目目录，光标闪烁等待输入。
