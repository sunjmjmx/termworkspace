=== 场景：项目选择对话框 ===
文件：06-project-dialog.png（待补充真实截图）

触发方式：首次启动时自动弹窗，或启动时无已保存的项目路径

对话框特征：
- 原生 macOS 文件选择对话框
- 标题：「选择项目文件夹」
- 选中目录后终端自动 cd 到该目录
- 文件树自动展开所选目录

交互流程：
1. 对话框弹出 → 列出最近使用的目录
2. 用户选择一个代码目录 → 点「打开」
3. 对话框关闭 → 终端自动 cd + 文件树展开
4. 下次启动自动恢复，不再弹窗

配置存储：选择的路径保存到 ~/.termworkspace/app-config.json

测试命令：rm ~/.termworkspace/app-config.json → 重启应用
测试结果：✓ 删除配置后重启，重新弹出选择对话框
